localStorage 객체

 - getItem(키) : 저장된 값을 추출.
 - setItem(키) : 값을 저장.
 - removeItem(키) : 특정 키의 값을 제거.
 - clear() : 모든 값을 제거.

 localStorage 객체는 웹 브라우저가 제공해주는 기능. (웹 API 라고 한다.)

ex) 모질라 브라우저에서는

배터리 정보를 추출하는 API, 블루투스, 2D/3D 처리 캔버스, 게임 패드 입력, 위치 정보, 드래그앤드롭, 이미지 동영상 캡쳐, 네트워크 정보, 웹 오디오 미디어 처리 등등의 수 많은 API들이 존재한다.

HTML5 시대부터는 웹 브라우저에서 미디어 처리 / 실시간 동영상 스트리밍, 3D 게임도 돌릴 수 있게 되었다.
